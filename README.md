# Embedded Thermal Alarm System

## Project Hierarchy Description

- `master_ref`
  - Primary Reference Manual (Reg-Level)
  - Primary Microcontroller Header (Reg-Level)
- `main.c`
  - mother of all application implementation
- `drivers_inc`
  - All function and macro definitions for external interfacing
- `drivers_src`
  - External Interface C implementation
- `clocking_inc`
  - All function and macro defintions for internal clock control
- `clocking_src`
  - implementation of internal clocking control
- `app_inc`
  - function and macro definitions for application functionality
- `app_src`
  - implementation of application peripherals
- `int_hand_inc`
  - definitions used for interrupt handling
- `int_hand_src`
- `linker script` _(coming soon; ends in `.ld`)_
- `Makefile`
